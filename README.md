# EqualVoice

A focused safety and incident reporting platform for gender-based harassment, discrimination, abuse, and workplace/school inequality.

## Project structure
- `index.html` — Home page
- `report.html` — Guided incident report workflow
- `my-reports.html` — Report dashboard
- `support.html` — Support services and guidance
- `emergency.html` — Emergency contact system with geolocation support
- `assets/css/style.css` — Modern responsive styling
- `assets/js/app.js` — Client-side architecture and form logic
- `docs/` — architecture, roadmap, user flow, database schema, scalability plan

## Features
- Mobile-first layout with WCAG-friendly contrast
- Step-by-step guided reporting workflow
- Anonymous reporting option
- Local storage for saved reports
- Trusted contact alert prototype with geolocation handling
- Privacy and confidentiality messaging
- Backend-ready architecture for Firebase/Supabase
