# EqualVoice MVP Roadmap

## MVP Scope
- Core landing page with trust messaging
- Guided incident reporting workflow
- My Reports dashboard for saved records
- Support Services page with help options
- Emergency help page with geolocation and trusted contact alerts
- Local storage persistence and basic client-side validation
- Accessible, mobile-first UI with WCAG-friendly contrast

## Phase 1: Immediate MVP
- Create `index.html`, `report.html`, `my-reports.html`, `support.html`, `emergency.html`
- Build `assets/css/style.css` and `assets/js/app.js`
- Implement multi-step report workflow
- Add anonymous reporting and evidence upload UI
- Build My Reports list and status badges
- Add emergency contact form and geolocation status

## Phase 2: Backend & Authentication
- Add Firebase or Supabase backend integration
- Add user authentication and session handling
- Store reports in encrypted database records
- Upload evidence files to secure storage
- Add email / SMS notifications for trusted contacts

## Phase 3: Trust, Legal, and Support
- Add privacy statement, confidentiality notes, data protection notice
- Add support directories for local hotlines and counseling
- Add case history export and safe sharing workflows
- Add role-based support for advocates and administrators

## Phase 4: Growth and Resilience
- Add multi-language and accessibility audits
- Add analytics for report trends and backlog heatmaps
- Add secure escalation workflows for critical incidents
- Add AI-assisted report summarization and recommendation refinement
